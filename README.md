# Python Django图书管理系统 python422

## 项目概述

本项目是基于Python的Django框架开发的图书管理系统。系统旨在为图书馆管理提供便捷、高效的信息化解决方案。

## 技术栈

- **后端：** Python 3.x，Django 3.x
- **前端：** HTML，CSS，JavaScript，Bootstrap
- **数据库：** SQLite（默认），支持其他数据库如MySQL、PostgreSQL等
- **其他工具：** pip，virtualenv

## 功能模块

- **用户管理：** 系统用户的注册、登录、权限管理
  - 用户认证
  - 角色权限控制
- **图书管理：**
  - 图书信息录入、编辑、删除
  - 图书分类管理
- **借阅管理：**
  - 图书借阅
  - 图书归还
  - 借阅记录查询

## 系统角色

- **系统管理员：** 管理用户、图书分类、图书信息等
- **普通用户：** 进行图书借阅和归还

## 运行环境

- **操作系统：** 具有Python运行环境的操作系统，如Linux、Windows、macOS
- **Python版本：** Python 3.x（建议3.8及以上版本）
- **依赖管理：** 使用pip进行依赖包安装

## 部署步骤

1. 克隆项目代码到本地
2. 创建Python虚拟环境并激活
3. 安装项目依赖：`pip install -r requirements.txt`
4. 配置数据库（如需更换数据库）
5. 运行迁移：`python manage.py makemigrations` 和 `python manage.py migrate`
6. 启动开发服务器：`python manage.py runserver`

- **易用性：** 界面友好，操作简便，易于上手。
